# D-WINE
와인 프로젝트 (6인)
기간 : 2021.10.11 ~ 2021.12.10  기획(38일) ,구현(21일)

# 프로젝트 상세내용

## 주제
```
* 와인쇼핑몰 웹 서비스
```

## 목적
```
1. 혼술, 홈파티가 많아지면서 와인의 수요가 증가하지만 높은 진입장벽으로 인해 와인을 즐기기에 어려움을 겪고 있습니다.

2. D-WINE은 와인 입문자들을 위해 상품의 다양한 정보를 제공하며
   날씨와 해시태그 기준으로 추천을 받을 수 있는 서비스를 기획하였습니다.
```
